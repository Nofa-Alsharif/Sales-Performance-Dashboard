# 📊 Sales Performance Dashboard

## 📌 Overview

This project is an interactive **Sales Performance Dashboard** built using Power BI, based on the Superstore dataset.
It provides a high-level overview of sales performance, profitability, customer activity, and regional distribution.



## 🎯 Objectives

* Monitor overall business performance
* Track sales and profit trends over time
* Analyze sales distribution across regions and categories
* Understand customer segments and shipping preferences



## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)

* **Total Sales:** 2.30M
* **Total Profit:** 286.4K
* **Total Orders:** 5009
* **Total Customers:** 0.79K



### 🔹 Sales Analysis

* **Sales Trend (Monthly):**
  Tracks how sales evolve over time to identify seasonality and growth patterns

* **Sales by Category:**
  Compares performance across:

  * Technology
  * Furniture
  * Office Supplies

* **Sales by Region:**
  Highlights regional performance:

  * West
  * East
  * Central
  * South



### 🔹 Customer & Distribution Insights

* **Sales by Segment (Donut Chart):**
  Shows contribution of:

  * Consumer
  * Corporate
  * Home Office

* **Shipping Mode Distribution:**
  Displays usage of:

  * Standard Class
  * Second Class
  * First Class
  * Same Day



## 🧠 Key Insights

* The **West region** has the highest sales performance
* **Technology category** generates the most revenue
* The **Consumer segment** contributes the largest share of sales
* **Standard Class shipping** is the most frequently used option



## 🛠️ Tools & Technologies

* Power BI
* DAX (for measures and calculations)
* Data visualization and dashboard design
